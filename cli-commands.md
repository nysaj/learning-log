# コマンドライン(CLI)の学習記録

## ディレクトリ（フォルダ）の移動・確認
* `pwd`
    - カレントディレクトリ（今いるフォルダ）の絶対パスを表示
* `cd [ディレクトリパス]`
    - 指定したディレクトリへ移動
* `cd (パス指定無し)`
    - ホームディレクトリへ移動
* `cd ..`
    - 一つ上の階層（親フォルダ）へ移動
* `cd ../..`
    - 2つ上の階層（親の親フォルダ）へ移動
* `cd -`
    - 直前のディレクトリへ移動（戻る）

## ディレクトリ（フォルダ）、ファイルの作成・確認
* `mkdir [ディレクトリ名]`
    - 今いる場所に新しくディレクトリ（フォルダ）を作成
* `touch [ファイル名]`
    - 新しいファイルを作成する
* `ls`
    - カレントディレクトリ（今いるフォルダ）のファイルとディレクトリの一覧を表示
* `ls -a`
    - カレントディレクトリ（今いるフォルダ）の全てのディレクトリとファイルを表示（隠されたファイル、フォルダを含む）
* `ls -la`
    - カレントディレクトリ（今いるフォルダ）の全てのディレクトリとファイルを、作成した日時・サイズ付きで縦一列に表示
* `ls [ディレクトリパス]`
    - ディレクトリパスのディレクトリとファイルを表示

## エイリアス（alias）
* `alias [別名]='[コマンド]'`
    - コマンドに別名（エイリアス）を設定する（現在のターミナルのみ有効、閉じると解除される）

## echo
* `echo "[文字列]"`
    - 文字列を表示する
* `echo "[文字列]" >> [ファイル名]`
    - ファイルの末尾に文字列を追記する

例

```bash
echo "alias sail='bash vendor/bin/sail'" >> ~/.zshrc
```

`.zshrc` の末尾に `alias sail='bash vendor/bin/sail'` を追記する。

> `>>` はファイルの末尾に追記する。
